Tema 3 - grundlæggende animation

03.01.04 - link til scene, samt udvidet style tile

Denne opgave skulle indeholde vores valg af typografi, farver, former, vores UI-elementer, samt præsentation af spillets ide. Denne første uge, var især design præget. Vi blev introduceret til Adobe Illustrator, samt hvordan man eksporterer SVG-filer. Det var også i denne uge vi ide-genererede, skitsetegnede og valgte karakterdesign samt scenografi. Vi udviklede også vores aktivitetsdiagram og statemachine diagram, som skulle klargøre vores ide til spillet. Ugen slutter dog af med, at vi lærte hvordan vi skulle kode en scene og placere diverse spil-elementer, så vi var klar til næste uges kode forløb. Det var her vi stiftede kendskab med CSS positionering.

03.03.03 - Animation-opgaven

I dette forløb skulle vi producere vores eget interaktivt spil. Det var her vi blev introduceret til Javascript programmering, som skulle kreere vores animationer. Javascript skal automatisere det vi ellers kan gøre manuelt i css og html. Hele forløbet var opbygget med små del-opgaver, hvor vi lærte mere og mere om Javascript's muligheder, som til sidst skulle være med til at færdiggøre vores eget interaktive spil. I Javascript til mit spil bliver der både benyttet if else, variabler samt timer. Udover det, skulle vi selv sørger for musik og lydeffekter til spillet, og det var her at vi arbejdede i Adobe Audition.

03.05.01 - projekt: sex og samfund

Denne opgave var en gruppeaflevering, og der blev her forventet at vi benyttede al den viden og de værktøjer vi havde lært de foregående uger. Vi blev dog introducerede til et nyt projektstyringsværktøj, Scrum samt trello, som blev brugt på daglig basis. Opgaven gik ud på at skabe et spil for Privatsnak.dk (Sex og Samfund), som skulle fokusere på hævnporno og deling af nøgenbilleder. Min rolle i dette forløb, var primært design delen, da dette ikke var min stærkeste side. Kodning fra de foregående uger, havde sat fast og gav god mening, så jeg fandt design og illustrator delen relevant for mig, så jeg kunne forbedre min svagheder. I slutningen på den første af de to uger, testede vi vores prototype på en dreng i den specifikke målgruppe, dette blev gjort med 5 sekunders test af vores forside og derudover skulle vores testpersonen afprøve spillet og svare på relevante spørgsmål.
